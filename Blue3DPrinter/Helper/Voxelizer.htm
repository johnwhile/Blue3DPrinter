This tool allows you to convert an obj file into a blueprint converting the surface with blocks. The only usefulness is to have a sort of base to build the ship maintaining the correct proportions.

Bluepint Size: block length of the vessel, the game is "limited" to 250 blocks per side, but remember that 250x250x250 = 15,625,000 possible blocks ! The XYZ coordinates are as Directx Left-Handle Y-up, in game are show as green for Y, blue for Z and red for X.

Force central simmetry: For a better conversions of symmetrical vessels you can check this option, currently only one symmetry at a time is implemented. The simmetry's plane is in the middle of the side, so make sure your 3d model has a symmetry in the center to match correctly.

3d Model Filling: the way your 3d model fills the dimensions you have previously given.

BlockID: Please only use the number 403 which corresponds to the hull cube.
