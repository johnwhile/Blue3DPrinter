"AssetStudio.dll" : https://github.com/Perfare/AssetStudio. Library to read bundles files, it's not fast but it's necessary to read the 3D geometries and build the model.

"modelStorage.data": contain for this "Beta" version the estracted geometry from "...\Bundles\models" and "...\Bundles\shapes". it's a temporary solution, so I preferred to extract only the geometries that are 90% used by your models, for the next versions I'm going to extract everything.
WARNING: if you delete this file the tool ask you to rebuild the file, but it take about 5min due to poor performance of AssetStudio library.

"Common.dll" : my math and tool library used for other projects

"FbxSdkWrapper.dll" and "FbxTool.dll" :  They are my C++/CLI wrapper for FBX SDK 2020.2 since there is no a C# library on the internet.

"libfbxsdk.dll": from FBX SDK, compiled for x86-Release configuration.

"ICSharpCode.SharpZipLib.dll": C# library for Zip, GZip, Tar and BZip2.
